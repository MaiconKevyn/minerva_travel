## Why

Destination-specific language tips can make the guide more useful before and during travel, especially for children. The content should adapt to destination language and child age instead of adding generic phrases.

## What Changes

- Add destination language learning content to generated guides when a destination has a relevant local language.
- Adapt phrases and activities to child age bands and mixed-age families.
- Place language content in pre-trip, during-trip, and post-trip moments where useful.
- Keep language content optional and safe when language data is unknown or the destination uses the family's language.

## Capabilities

### New Capabilities

- `destination-language-learning`: Covers destination language detection, age-aware phrases, activities, and guide placement.

### Modified Capabilities

- None. No main specs exist yet in this repository.

## Impact

- Guide content planning and rendering.
- Destination metadata or language mapping utilities.
- Activity generation logic for age bands.
- Tests for language selection, age complexity, guide placement, and fallback behavior.
